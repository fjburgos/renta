## Convención

Cada especificación debe incluir:
1. **Objetivo** — qué calcula o resuelve
2. **Base normativa** — artículo de la Ley 35/2006 IRPF u otro texto legal
3. **Algoritmo** — descripción paso a paso del cálculo
4. **Ejemplos** — casos de uso con valores concretos
5. **Casos límite** — situaciones edge a contemplar
